## 🔧 CONFIGURATION OPTIONS

### Environment Variables (.env)
All agents use your existing configuration:
- `DISCORD_BOT_TOKEN`: Bot authentication
- `NOTION_SECRET`: Database access
- `OPENAI_API_KEY`: GPT-4 integration for smart responses
- `DISCORD_JOB_BOARD_CHANNEL_ID`: Primary operations channel

### Agent Customization
Each agent includes configurable thresholds:
- **Maya**: Praise frequency, milestone definitions
- **Zara**: Bonus rates, performance thresholds
- **Nikolai**: SOP requirements, photo minimums
- **Iris**: Pricing multipliers, discount limits
- **Jules**: Report schedules, KPI targets
